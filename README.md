# Nodenomics

**A Real-Time Economy for Connected Devices**

## Overview

Nodenomics enables machines to buy and sell data automatically, in real time, using tiny pay-per-use micropayments. Our platform creates a "data on demand" marketplace where devices can request exactly the data they need, receive it instantly, and pay only for what they use.
